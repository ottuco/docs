# Real Estate

Ottu's Real Estate Plugin streamlines property management and rental transactions for premises owners. Seamlessly integrated with Ottu's online payment management system, it provides a centralized dashboard for managing property portfolios, tenants, contracts, invoices, maintenance, and financial transactions.

## Key Features

- **Dashboard Access** -- Each merchant receives exclusive dashboard credentials for secure, centralized management of their property portfolios and rental transactions.
- **[Property Management](/business/industry/real-estate/properties)** -- Add new properties, update property details, and monitor occupancy status directly from the dashboard.
- **[Tenant Management](/business/industry/real-estate/tenant-contract/tenant-management)** -- Onboard new tenants, manage lease agreements, and oversee tenant-related transactions.
- **[Invoice Monitoring](/business/industry/real-estate/invoices-management)** -- Monitor invoices and track payment transactions in real-time, ensuring timely rental payments and financial transparency.
- **[Contract Management](/business/industry/real-estate/tenant-contract/contract-management)** -- Terminate contracts, perform manual transactions, adjust rental prices, and request advance payments from the dashboard.
- **[Reports & Analytics](/business/industry/real-estate/invoices-management#generate-report)** -- Generate reports on upcoming contract expirations and contract summaries for informed decision-making.
- **[Maintenance](/business/industry/real-estate/maintenance)** -- Streamline issue reporting and resolution with seamless communication between tenants and landlords.

## Regular Activities

The Real Estate Plugin supports a range of routine operations essential for efficiently managing properties, tenants, and financial workflows. These activities include:

| Activity | Description |
|---|---|
| [Property Management](/business/industry/real-estate/properties) | Add, edit, and manage property details and units |
| [Tenant & Contract Management](/business/industry/real-estate/tenant-contract/) | Manage tenants and their rental agreements |
| [Generate Invoices](/business/industry/real-estate/generate-invoice) | Create and send invoices with payment links |
| [Invoice Management](/business/industry/real-estate/invoices-management) | Track payments, send reminders, and generate reports |
| [Maintenance](/business/industry/real-estate/maintenance) | Report and resolve property maintenance issues |
| [Transactions](/business/industry/real-estate/transactions) | Monitor all payment transactions and balances |
| [Auditing](/business/industry/real-estate/auditing) | Track changes and roll back operations when needed |

By streamlining these activities, property managers and administrators can ensure seamless operations, improve tenant satisfaction, and maintain financial transparency.
