# Tenant Management

## Add New Tenant

To add a new tenant, click the **+ADD TENANT** button within the Tenant and Contract Management section.

![Add tenant button](/img/business/real-estate/tenant-add-button.png)

### New Tenant Form

The form fields vary depending on the tenant type.

#### For Residential Tenant Type

| Field | Description |
|---|---|
| **Tenant Name (EN)** | Name of the tenant in English |
| **Tenant Name (AR)** | Name of the tenant in Arabic |
| **Civil ID Number** | Civil ID number of the tenant |
| **Upload Civil ID (Front)** | Scanned copy of the front side of the civil ID |
| **Upload Civil ID (Back)** | Scanned copy of the back side of the civil ID |
| **Passport Number** | Passport number of the tenant (if applicable) |
| **Upload Passport** | Scanned copy of the passport (if applicable) |
| **Nationality** | Nationality of the tenant |
| **Phone Number** | Phone number of the tenant |
| **Email Address** | Email address of the tenant |
| **Select Notification Language** | Preferred language for notifications |
| **Is Married** | Whether the tenant is married (Yes/No) |

![Residential tenant form](/img/business/real-estate/tenant-residential-form.png)

**When the tenant is married**, additional fields appear for the spouse's information:

| Field | Description |
|---|---|
| **Civil ID Number** | Civil ID number of the spouse |
| **Upload Civil ID (Front)** | Scanned copy of the front side of the spouse's civil ID |
| **Upload Civil ID (Back)** | Scanned copy of the back side of the spouse's civil ID |
| **Passport Number** | Passport number of the spouse |
| **Upload Passport** | Scanned copy of the spouse's passport |
| **Nationality** | Nationality of the spouse |
| **Phone Number** | Phone number of the spouse |
| **Email Address** | Email address of the spouse |
| **Upload Marriage Certificate** | Scanned copy of the marriage certificate |

![Spouse information form](/img/business/real-estate/tenant-spouse-form.png)

#### For Limited Liability Company or Listed Company Tenant Type

| Field | Description |
|---|---|
| **Tenant Name (EN)** | Name of the company in English |
| **Tenant Name (AR)** | Name of the company in Arabic |
| **Civil ID Number** | Civil ID number of the company representative |
| **Commercial Registration Number** | Commercial registration number of the company |
| **Phone Number** | Phone number of the company representative |
| **Email Address** | Email address of the company representative |
| **Select Notification Language** | Preferred language for notifications |
| **Upload Commercial Registration** | Scanned copy of the commercial registration |
| **Upload Authorized Signatory Confirmation** | Confirmation document for the authorized signatory (Listed Company only) |
| **Upload Trade License** | Scanned copy of the trade license |
| **Upload Authorized Signatory Name** | Name of the authorized signatory |
| **Upload Attachment 1-3** | Additional attachments (if applicable) |

![Company tenant form](/img/business/real-estate/tenant-company-form.png)

## Edit Tenant

1. **Search for the Tenant** -- Locate the desired tenant from the list.
2. **Open the Dropdown Menu** -- Click the three dots (**...**) at the beginning of the tenant's row.
3. **Select Edit Tenant** -- Choose **Edit Tenant** from the dropdown menu to open a pre-filled form.
4. **Update Details** -- Modify the editable fields as needed.
5. **Save Changes** -- Click **Save** to apply the changes.

![Edit tenant dropdown](/img/business/real-estate/tenant-edit-dropdown.png)
