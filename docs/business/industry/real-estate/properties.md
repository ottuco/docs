# Property Management

This section covers adding, editing, and managing property details within the Ottu Real Estate Plugin. It ensures accurate property records and optimized workflows for seamless property administration.

## Property Management Dashboard

The **Property Management Dashboard** provides a clear overview of all properties and easy access to essential property details and actions.

**Key features:**

- **Property Overview** -- View a list of all properties with addresses, contracts, and unit statuses.
- **Quick Access** -- Easily view property details like classification, reporting profile, and invoice profile.
- **Unit Status** -- Track available, rented, and under-maintenance units at a glance.
- **Filter Options** -- Quickly find properties using filters for location, availability, and classification.
- **Add & Edit Properties** -- Directly add, edit, or update property and unit details.
- **Reports** -- Generate PDF reports on unit vacancy and occupancy status.

## Add New Property

To add a new property, navigate to the **Property Management** section and click the **+ADD PROPERTY** button.

![Add property button](/img/business/real-estate/placeholder.png)

You will be redirected to the form for adding a new property.

### New Property Form

Fill out the following fields:

| Field | Description |
|---|---|
| **Property Name (EN)** | The name of the property in English |
| **Property Name (AR)** | The name of the property in Arabic |
| **Address** | The physical address of the property |
| **Province** | The province where the property is located |
| **Town** | The town or city where the property is situated |
| **Property Classification** | The classification or category of the property |
| **Reporting Profile** | Groups multiple properties under one profile for consolidated reporting |
| **Invoice Profile** | Manages the logo, notification, and signature for receipts and checkout pages associated with this profile |
| **Gateway** | The payment gateway associated with the property |
| **Allow Cash Payment** | Option to enable or disable cash payments (check or uncheck) |

![New property form](/img/business/real-estate/placeholder.png)

### Add Unit Types and Units

To add multiple unit types (e.g., villas and apartments) along with their respective units:

1. Click **ADD UNIT TYPE WITH UNIT** to add a unit type and new units.
2. **Unit Type**: Choose from existing unit types in the dropdown list, or add a new one by clicking **ADD UNIT TYPE** and filling in the required details.

![Add unit type](/img/business/real-estate/placeholder.png)

3. **Add Units**: Click the **ADD OR EDIT UNITS** button.

![Add or edit units](/img/business/real-estate/placeholder.png)

4. **Define the New Unit**: After specifying the number of units and their prices, click the **ADD** button.

![Define new unit](/img/business/real-estate/placeholder.png)

5. **Unit Details**: Provide details such as **Unit Number** and **Floor** for each unit.

![Unit details form](/img/business/real-estate/placeholder.png)

Repeat the above steps until all property unit types are added. Then click the **Save** button in the New Property Form to save all changes.

![Save property with units](/img/business/real-estate/placeholder.png)

## Edit Property

Merchants can update property details through a straightforward process:

1. **Access Property List** -- Go to the **Property Management** section.
2. **Select Property** -- Click on the **property name** you want to edit.
3. **Edit Form Opens** -- A pre-filled form with the property's details appears.
4. **Modify Details** -- Update property information as needed.
5. **Manage Units** -- Edit existing units or add new units if required.
6. **Save Changes** -- Finalize updates and save.

## Generate Property Report

Merchants can generate a PDF report detailing property vacancy and occupancy.

### How to Generate the PDF Report

1. **Access the Property Management Table** -- Navigate to the **Property Management** section in your dashboard.
2. **Define Filter Parameters (Optional)** -- Click the **Filter** button and set your desired criteria.
3. **Click the Print Button** -- From the Property Management table, click the **Print** button.

![Print property report](/img/business/real-estate/placeholder.png)

4. **Automatic PDF Generation** -- The PDF generates automatically after clicking Print.
5. **Download the PDF File** -- Once generated, download it to your device.

![Generated property report](/img/business/real-estate/placeholder.png)
