# Auditing & Rolling Back Activities

Auditing and rollback activities are essential for ensuring transparency, accountability, and control over key contract-related operations. These features allow administrators to track changes, reverse unintended modifications, and maintain data integrity across various contract scenarios.

:::note
All the lists below are accessible from the **Dashboard** under the **Action Listings** section.
:::

## New Contract List

This section displays all [contracts](/business/industry/real-estate/tenant-contract/add-contract), providing basic information such as:

- ID
- Creation date
- Initiator
- Tenant name
- Property
- Unit number
- Contract start and end dates
- Contract reference

Merchants can access further details by clicking the dropdown arrow at the end of each contract row.

### Contract Rollback

To roll back a new contract, click on the three dots (**...**) at the end of the contract row and select **Rollback**.

![Contract rollback action](/img/business/real-estate/placeholder.png)

## Renew Contract List

This list displays the same details and functionality as the New Contract List, specifically for contracts that have been [renewed](/business/industry/real-estate/tenant-contract/renew-contract).

### Renew Contract Rollback

To roll back a renewal contract operation:

1. Navigate to the Renew Contract List.
2. Locate the desired contract.
3. Click the **three dots** (**...**) at the end of the row.
4. Select **Rollback** from the dropdown menu.

![Renew contract rollback](/img/business/real-estate/placeholder.png)

## Terminate Contract List

This list presents basic information for [terminated contracts](/business/industry/real-estate/tenant-contract/terminate-contract), including:

- ID
- Creation date
- Initiator
- Tenant name
- Property
- Unit number
- Contract start and end dates
- Termination date

Merchants can access additional details for each contract by clicking the dropdown arrow.

### Terminate Contract Rollback

To undo a termination:

1. Go to the Terminate Contract List.
2. Find the specific contract.
3. Click the **three dots** (**...**) at the end of the row.
4. Select **Rollback** from the dropdown menu.

![Terminate contract rollback](/img/business/real-estate/placeholder.png)

## Manual Payment List

All [manual payments](/business/industry/real-estate/tenant-contract/manual-payment) are listed with the following details:

- ID
- Creation date
- Initiator
- Property name
- Unit number
- Due date
- Amount

Each row represents one manual payment.

### Manual Payment Rollback

To roll back a manual payment:

1. Navigate to the Manual Payment List.
2. Locate the specific payment.
3. Click the **Action Button** at the end of the row.
4. Select **Rollback** from the dropdown menu.

![Manual payment rollback](/img/business/real-estate/placeholder.png)

## Advance Payments List

All [advance payment](/business/industry/real-estate/tenant-contract/advance-payment) requests are listed with key details:

- Contract ID
- Creation date
- Initiator
- Tenant name
- Property
- Unit number
- Contract dates
- Contract reference
- Advance amount

Clicking on a row opens a popup with two tabs:

- **Contract Tab** -- Displays detailed information about the selected contract.
- **Invoices Tab** -- Shows paid advance payments linked to the request.

![Advance payment details](/img/business/real-estate/placeholder.png)

:::note
One advance payment request can include more than one invoice.
:::

### Advance Payment Rollback

To roll back an advance payment:

1. Click the **three-dot action menu** at the end of the corresponding row.
2. Select **Rollback** from the dropdown options.

![Advance payment rollback](/img/business/real-estate/placeholder.png)

## Modified Contract List

All modified contracts are displayed in the Modified Contract List. This section provides an organized view of contracts that have undergone changes, allowing merchants to track and review modifications.

![Modified contract list](/img/business/real-estate/placeholder.png)
